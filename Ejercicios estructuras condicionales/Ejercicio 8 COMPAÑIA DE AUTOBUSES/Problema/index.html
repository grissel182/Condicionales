Ejercicio #8: COMPAÑIA DE AUTOBUSES

Una compañía de viajes cuenta con tres tipos de autobuses (A, B y C), cada uno tiene un precio por kilómetro recorrido por persona, los costos respectivos son $2.0, $2.5 y $3.0.
Se requiere determinar el costo total y por persona del viaje considerando que cuando éste se presupuesta debe haber un mínimo de 20 personas, de lo contrario el cobro se realiza con base en este número límite

Pseudocodigo

Variable Descripción Tipo
A Autobus que cobra $2.00 por km Int / Float
B Autobus que cobra $2.50 por km Int / Float
C Autobus que cobra $3.00 por km Int / Float
P Número de personas de viaje Int
Km Número de kilometros a recorrer Int / Float
Min 20, número mínimo de personas Int
Costo Km por precio seleccionado, costo unitario Int /Float
Total Costo total del camion (por todas las personas y todo el kilometraje) Int / Float
  1. Inicio
  2. Leer A, B, C, Km, P, Min
  3. Multiplicar Km * A, B, o C = Costo
  4. Multiplicar el Costo¨* P = Total (el # min de personas es 20)
  5. Mostrar el Costo unitario y costo total
  6. Fin
